List of chat commands on twitch.tv/khaotico

  • !twitter - link to Khaotico's twitter
  • !cinch - Khaotico's 5% off CinchGaming.com code "KHT"
  • !chat - when the spam gets to be too much
  • !feels - for use when Khaotico's stream is ending
  • !time - current time for Khaotico
  • !uptime - how long the stream has been up for
  • !lucio - lit catchphrases by Lucio
  • !mercy - Heroes never die
  • !reaper - edgy sayings
  • !mccree - just about everyone's reaction
  • !pharah - give that man some justice
